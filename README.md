# 3d-point-visualizer
Simple project for visualizing the position of a point using three.js

## Dev
### Prerequesites

- Node > 9.x
- NPM > 6.x

Note: Might work on lower versions

### Install dependencies
Run
```bash
npm i
```

### Start dev
Run
```bash
npm run dev
```

### Build production build
Run
```bash
npm run build
```
